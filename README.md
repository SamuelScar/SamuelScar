## Selected projects

### [SIGL — Interactive Group Management System](https://sigl.nsafr.tech)

The most complete product I have independently designed and developed so far.

SIGL is a robust full-stack web platform for creating, organizing and conducting interactive group activities. The project covers the complete product lifecycle, including problem analysis, requirements definition, architecture, interface design, backend development, real-time communication, testing, security, infrastructure and production deployment.

The system includes authentication, different user roles, room access through unique codes, configurable activity flows, real-time communication and interactive controls for managing sessions and participant experiences.

The application runs in a production environment on a Linux VPS using Docker, HTTPS, reverse proxy configuration, domain management, security controls and an organized deployment workflow.

The source code is currently maintained in a private repository because the project is still under active development.

**Highlights:** Full product development · Real-time communication · Authentication and authorization · Business rules · Docker · Linux VPS · Production deployment

### [ANS Despesas Analytics](https://github.com/SamuelScar/ans-despesas-analytics)

Data engineering and backend application developed with Python for ingesting, validating, enriching, storing and analyzing public expense data from the Brazilian National Supplementary Health Agency.

The project includes a complete ETL pipeline, PostgreSQL persistence, analytical SQL queries and a FastAPI application organized with Router, Service and Repository layers.

It also documents architectural decisions, inconsistencies found in the original datasets, technical trade-offs and the strategies adopted to process heterogeneous public files efficiently.

**Highlights:** Python · FastAPI · ETL · PostgreSQL · Data validation · Layered architecture

---

### [CEASA Data Platform](https://github.com/SamuelScar/ceasa-statistics)

Data collection, processing and analytical solution for public price quotations from Brazilian CEASAs.

The platform combines automated data extraction, normalization, relational storage and an interactive analytical dashboard with filters, indicators, charts, aggregated queries and database-level pagination.

The dashboard was developed with Python and Streamlit, while PostgreSQL and Supabase are used for data persistence. Docker provides a reproducible execution environment.

**Related repository:** [CEASA Data Scraper](https://github.com/SamuelScar/cotacoes-ceasa-scraper)

**Highlights:** Web scraping · Data normalization · PostgreSQL · Streamlit · Docker · Data visualization

---

### [VitaFlow](https://github.com/SamuelScar/VitaFlow)

Web system developed with Laravel for managing blood donation campaigns, collection locations, schedules, appointments and participation reports.

The project demonstrates the implementation of business rules, relational data modeling, administrative workflows and a complete web application using the Laravel ecosystem.

**Highlights:** PHP · Laravel · Business rules · Relational database · Web application

### [Instrumentos do Mundo Bíblico](https://instrumentos-biblicos-ar.vercel.app/)

Interactive educational web application that connects biblical studies, history, music and sound science through multimedia content and three-dimensional models.

The platform includes a responsive catalog, educational pages, historical and scientific content, interactive 3D visualization, audio demonstrations, light and dark themes and augmented-reality experiences for compatible devices.

The project was developed with Vue 3, TypeScript, Vite, Three.js-related technologies, Docker and Nginx, with static deployment through Vercel.

**Repository:** [instrumentos-biblicos-ar](https://github.com/SamuelScar/instrumentos-biblicos-ar)

**Highlights:** Vue 3 · TypeScript · 3D visualization · Augmented reality · Responsive design · Docker
