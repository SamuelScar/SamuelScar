## About me

I am a backend-focused developer with around five to six years of experience with **PHP**, building web applications, APIs, integrations and data-oriented solutions.

My strongest stack is **PHP, Laravel and Slim**, but I also develop applications and services with **Node.js, Express and Python**. Recently, I have been using Python extensively for web scraping, ETL pipelines, data normalization, dashboards, automation and numerical analysis.

I participate in the complete software development process, including requirements analysis, user stories, use cases, architecture decisions, implementation, automated testing, API documentation, maintenance and code review.

Although backend development is my main area, I also have practical experience with frontend technologies such as **HTML, CSS, JavaScript, jQuery, Bootstrap, Tailwind CSS and Laravel Blade**.

I am currently focused on academic, personal and open-source projects, while looking for opportunities involving backend development, APIs, integrations and data processing.

## Selected projects

### [CEASA Statistics](https://github.com/SamuelScar/ceasa-statistics)

Interactive dashboard developed with Python and Streamlit for analyzing public price quotations from Brazilian CEASAs, using filters, charts, indicators and Docker.

### [VitaFlow](https://github.com/SamuelScar/VitaFlow)

Web system developed with Laravel for managing blood donation campaigns, collection locations, schedules, appointments and participation reports.

### [ANS Despesas Analytics](https://github.com/SamuelScar/ans-despesas-analytics)

ETL pipeline and analytical application developed with Python for ingesting, validating, enriching and analyzing public expense data from the Brazilian National Supplementary Health Agency.

### [BlogSharp](https://github.com/SamuelScar/BlogSharp)

Backend application developed with ASP.NET Core for a personal blog, featuring authentication, data persistence, layered architecture and integration with artificial intelligence resources.

### [Kanban Board Web](https://github.com/SamuelScar/kanban-board-web)

Web application for organizing tasks and managing workflows through a Kanban-style interface.

### [Binário em Palavras](https://github.com/SamuelScar/Binario-em-Palavras)

Educational application that converts text to binary and binary to text, demonstrating how computers represent and process information.
